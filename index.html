Instruction Types
Memory-reference

Interacts with the Memory Unit. The operand specifies a memory address to read from or write to.

Opcode  +  Memory Address
LOAD   STORE   ADD   SUB   JMP
Register-reference

Operates on data already inside CPU registers. No memory access needed — the operand holds register control bits.

Opcode  +  Register control bits
AND   OR   NOT   HALT   NOP
I/O instruction

Controls communication with external devices like keyboards and displays. The operand holds I/O control bits.

Opcode  +  I/O control bits
IN   OUT
